## Installation

```lua
use({
  'davidsierradz/cmp-conventionalcommits',
  branch = 'feature/addYarnWorkspacesToScopes',
  rocks = { { 'semver' } },
})
```

## Setup

Setup in `after/ftplugin/gitcommit.lua`

```lua
require'cmp'.setup.buffer {
  sources = require'cmp'.config.sources(
    {{ name = 'conventionalcommits' }},
    {{ name = 'buffer' }}
  ),
}
```
